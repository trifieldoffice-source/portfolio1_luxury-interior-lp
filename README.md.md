# Webサイト納品データ一式

本フォルダには、〇〇様（クライアント名）の公式Webサイトを構成するすべてのデータが格納されています。
本データは、お客様の「デジタル資産」として大切に保管していただきますようお願い申し上げます。

---

## 1. データの構成とその役割について

本パッケージは、大きく分けて以下の2つのデータで構成されています。

### ① 【公開用データ】（`out` フォルダ内）
- **役割**: 実際にインターネット上で公開し、閲覧者に見せるための完成済みデータ（HTML/CSS/JavaScript）です。
- **使い方**: この `out` フォルダの中身を、ご契約のレンタルサーバーやホスティングサービス（Vercel, AWS, Xserverなど）にそのままアップロードしていただくだけで、即座にサイトが公開・表示されます。

### ② 【編集用ソースコード】（`src` フォルダ等を含むその他の全ファイル）
- **役割**: サイトの設計図となる、専門的なプログラムの原本データです。
- **使い方**: 今後、文字の修正、画像の差し替え、新たなセクションの追加など「サイトの更新・改修」を行う際に、エンジニアや制作会社が使用します。

---

## 2. 万が一の際の「再構築の保証」について

私（制作者）は、本サイトを特定の個人や会社がいなければ修正できない「ブラックボックス（囲い込み）」の状態にはしていません。

本データ一式をお手元に保管していただいている限り、**仮に将来、私と連絡が取れなくなるような不測の事態が発生したとしても、この元データを他のWeb制作会社やエンジニアにお渡しいただければ、問題なくサイトの再構築や引き継ぎ、更新作業を行うことが可能**です。

これは、世界中で標準的に使われている最新かつオープンな技術を採用して構築しているためであり、お客様のビジネスリスクを最小限に抑えるための強いこだわりです。

---

## 3. 本サイトの技術的優位性（専門的な補足）

本サイトは、簡易的なホームページ作成ツール（WordPressやWixなど）を使用せず、世界最先端のフロントエンド技術である **「Next.js」** および **「Framer Motion」** を用いてフルスクラッチ（ゼロからの手組み）で構築されています。これにより、以下の圧倒的なメリットを実現しています。

1. **極めて高いセキュリティ**: 
   データベースを持たない「静的サイト（Static Site）」として書き出されるため、ハッキングや改ざんのリスクが極限まで低減されています。
2. **圧倒的な表示速度**: 
   ページが事前に最適化されて生成されているため、ユーザーを待たせない高速な読み込みを実現し、SEO（検索エンジン対策）にも大きく寄与します。
3. **滑らかで高級感のあるアニメーション**: 
   Apple社のサイトなどでも採用される高度なアニメーション技術（Framer Motion）を使用し、「ただ要素が動く」だけでなく、物理法則に基づいた優雅で無重力感のある、ブランドの品格を高める演出を実装しています。
